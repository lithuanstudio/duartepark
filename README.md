# Duarte Park Estacionamento

Site institucional estático em HTML, CSS e JavaScript.

## Ajuste desta versão

A seção **Trabalhe Conosco / Contrate para Eventos** foi ajustada no código para seguir o print enviado:

- dois cards lado a lado;
- card branco com leve tom amarelado;
- borda fina amarela clara;
- ícone circular amarelo;
- título em duas linhas;
- palavra principal em amarelo;
- botão amarelo pequeno;
- sem foto dentro dos cards;
- seção posicionada abaixo do “Quem Somos”, antes da parte de tecnologia.

## Como rodar

Abra `index.html` no navegador ou suba a pasta na Vercel.

## Onde trocar dados

No arquivo `index.html`, procure por:

- `5511999999999` para trocar o WhatsApp/telefone;
- `contato@duartepark.com.br` para trocar o e-mail;
- endereço em `localizacao` para trocar a localização.
